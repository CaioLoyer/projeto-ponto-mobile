# Ponto Mobile

**Alunos:** Caio Loyer e Murilo Toledo

## Objetivo

A aplicação tem como objetivo simular um sistema de registro de ponto para empresas. Sua principal funcionalidade é coletar as informações de registro de ponto de um colaborador — **nome, horário do registro, localização e imagem** — e armazená-las em um banco de dados para posterior consulta.

## Execução

```bash
pnpm install
pnpm start# Ponto Mobile
