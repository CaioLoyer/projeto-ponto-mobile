# Ponto Mobile

**Alunos:** Caio Ricarte Loyer e Murilo Toledo Vieira

## Contexto

A ideia do desenvolvimento do aplicativo, é trazer uma abordagem simples e intuitiva para registrar a presença de colaboradores no ambiente de trabalho. Mas que forneça precisão e segurança para que não ocorram fraudes ou manipulações nos dados informados.

## Objetivo

A aplicação possui o objetivo de simular uma aplicação de registro de ponto para empresas.
Tendo como a principal funcionalidade coletar as informações de registro do ponto de um colaborador(nome, horário de registro, localização e imagem), gravar no banco de dados para listagem e comparar com a localização da empresa cadastrada para validar se o ponto foi registrado dentro do local determinado.

## Execução

```bash
pnpm install
pnpm start# Ponto Mobile
```
